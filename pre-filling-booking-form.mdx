## What This Does

When a website visitor fills out your custom form, you can:

1. Send them to a **lunacal scheduling booking page** OR
2. Show your **lunacal embedded scheduler**

â€¦and have fields like **name, email, phone, notes, or other custom fields** already filled in.

## How It Works

lunacal supports prefilling booking fields by passing values either: \\

1. via **URL query** **parameters** \
   OR\\
2. via **embed configuration** (if you embed lunacal directly on your site)

## 1. Prefill via URL Parameters (Scheduling Page & Embeded Page)

If your form submits to a landing page or redirects after submission, build the lunacal booking URL with query parameters.

ðŸ”— Example Redirect URL

> https://lunacal.ai/your-username/your-event?name=John%20Doe&email=john%40example.com&attendee-phone=%2B919876543210

## **You have to use booking questions identifier (Internal label ) as query params**

![](https://lunacalwebsite.blob.core.windows.net/website-assets/HelpDocs/prefill-1.webp)

### **Supported Prefill Fields**

**Example :**

| **Field**       | **Query Parameter ( Identifier )** |
| :-------------- | :--------------------------------- |
| Attendee name   | name                               |
| Attendee email  | email                              |
| Phone number    | attendee-phone                     |
| Comments        | notes                              |
| Custom question | yourCustomQuestionFieldIdentifier  |

{yourFieldIdentifier}

> Spaces must be URL-encoded (%20), and + in phone numbers must be URL-encoded as %2B.

## **2. Prefill in an Embedded lunacal Scheduler**

If you embed lunacal on your own page â€” inline or via a popup â€” you can pass prefill values directly in the embed configuration.

You donâ€™t need to use URL parameters in this approach.

## **You have to use booking questions identifier (Internal label ) as key**

### Inline Embed Example (JavaScript)

```
<script src="https://lunacal.ai/embed.js"></script>
<div id="lunacal-scheduler"></div>
<script>
  lunacal.init({
    target: "#lunacal-scheduler",
    config: {
      attendee-name: "John Doe",
      email: "[email protected]",
      phone: "+919876543210",
      notes: "Filled from website form",

      // Any custom metadata or questions
      "metadata[campaign]": "summer-camp"
    }
  });
</script>
```

### Floating Button Popup Embed

```
<script src="https://lunacal.ai/embed.js"></script>
<script>
```
