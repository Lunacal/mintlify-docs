## What This Does

When a website visitor fills out your custom form, you can:

1. Send them to a **lunacal scheduling booking page**.\
   OR

## How It Works

lunacal supports prefilling booking fields by passing values either:

1. via **URL query** **parameters** \
   OR
2. via **embed configuration** (if you embed lunacal directly on your site)

## **2. Prefill in an Embedded lunacal Calendar**

You can use the same query parameter method in the embed as well.

Alternatively, if you prefer not to use query parameters, you can configure the values directly in the embed settings.

## **You have to use booking questions identifier (Internal label ) as key**

### Inline Embed Example (JavaScript)

```
  Lunacal.ns["your-event-slug"]("inline", {
        elementOrSelector: "#your-event-slug",
        config: {
          layout: "",
          // Add your booking questions here.
		  // Use the questionâ€™s internal label as the key,
	      // and set the value to the prefill text you want to pass.

          "custom-question": "pre-fill-custom-question",

        },
        calLink: "your-event-slug",
      });
```
