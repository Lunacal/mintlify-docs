## Voice Agent For Meeting Reminders

Voice Agent Calls let you trigger an outbound AI call as part of a workflow after a meeting is booked.

The agent can call the attendee and help confirm the meeting, reschedule it, or cancel it

Voice Agent Calls are available inside Workflows.

---

## Current availability

Voice Agent Calls are currently available only for US, Australia, Canada and UK\
We expect this to be available for all countries by April 7, 2026.

---

## Pricing

Voice Agent Calls are priced at \$0.15 per minute.\
In the coming weeks, the pricing may vary slightly by country because outbound calling charges are different across destinations.\
Each call is capped at 3 minutes maximum.\
The same credits balance can be used for both Voice Agent Calls and SMS. This means your AI and messaging usage draw from the same

---

## What the Voice Agent can do

The Voice Agent can:

- remind the attendee of the meeting
- reschedule if they ask for it
- cancel if they ask for it
- provide basic details about the meeting

---

## How to create AI Call Workflow

- Go to the [Workflows](https://app.lunacal.ai/new-settings?tabName=workflows) tab and click **Create New Workflow**
- Provide a unique name for your workflow in the designated field.
- Determine the event type to which the workflow will apply.
- Choose when the workflow should be triggered, for now AI Call available for - **Before an event starts**
- Change send email to attendee to Send AI call to attendee.

Read this if you stuck when creating workflow : [https://help.lunacal.ai/sms-and-email-workflows](https://help.lunacal.ai/sms-and-email-workflows)
