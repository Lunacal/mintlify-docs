---
title: "Booking Questions"
description: "Use “Booking Questions” to collect the key details you need before a meeting. You’ll find this under the “Event Types” you want to configure."
mode: "wide"
---

**Step 1**

On “Event Types”, select the event type you want to edit. You can click anywhere on the square box or use the gear button.

<Frame>
  ![](/images/booking_questions/1.webp)
</Frame>

**Step 2**

Navigate to the **Booking Questions** tab.

<Frame>
  ![](/images/booking_questions/2.webp)
</Frame>

**Step 3**

**Note: **You can't delete default questions.

**Add/Edit your booking question**. When adding question you can select the question type : Email, Phone, Short text, Long text,  Dropdown, Multiple Emails, Check Box (Multi select), Multi Choice ( Single select), Pipedrive Custom field, Consent Checkbox, 

While adding question you can give [ Question Name, Internal label, Answer Suggestion ] 
<Frame>
  ![](/images/booking_questions/editingq.webp)
</Frame>

When Editing Name question you can use a feature called split to ask booker to put first Name and last name as separate

<Frame>
  ![](/images/booking_questions/3.webp)
</Frame>

> **<u>Please don't edit the default booking questions that you see. It might lead to unexpected behaviour. You can turn them off if you don't want to use it.</u>**

**Step 4**

Use the toggle to “Hide/Unhide” any question.

<Frame>
  ![](/images/booking_questions/4.webp)
</Frame>

**Step 5**

Turn on “Mandatory” to require an answer, then click “Save”.

<Frame>
  ![](/images/booking_questions/5.webp)
</Frame>
