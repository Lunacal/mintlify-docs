# Setting Up Booking Questions

In this article, you'll learn how to set up booking questions for your event. 

**Step 1:** Choose your event type and click on it.

<Frame>
  <img
    src="https://usercontent.in.prod.clueso.io/ebbf25da-f06e-4e91-8b05-b5f8e12127c5/e14daef3-e6cd-4de9-9854-6a86bc00408f/8424db4f-2c21-44ce-9816-315dcf1c3934/images/6fa3b3aa-e5cd-4efc-a13e-305464d32496.png"
    style={{ borderRadius: "0.5rem" }}
  />
</Frame>

**Step 2:** Navigate to the **Booking Questions** tab.

<Frame>
  <img
    src="https://usercontent.in.prod.clueso.io/ebbf25da-f06e-4e91-8b05-b5f8e12127c5/e14daef3-e6cd-4de9-9854-6a86bc00408f/8424db4f-2c21-44ce-9816-315dcf1c3934/images/af51f6fa-1256-4a9b-b0e5-1844c2e1f72d.png"
    style={{ borderRadius: "0.5rem" }}
  />
</Frame>

**Step 3:** Here, you're able to add or edit your booking questions. Click on it to proceed.

<Frame>
  <img
    src="https://usercontent.in.prod.clueso.io/ebbf25da-f06e-4e91-8b05-b5f8e12127c5/e14daef3-e6cd-4de9-9854-6a86bc00408f/8424db4f-2c21-44ce-9816-315dcf1c3934/images/befd0550-bac7-4f2f-a146-0ba36a7c57ae.png"
    style={{ borderRadius: "0.5rem" }}
  />
</Frame>

**Step 4:** Add your new question. For instance, we'll click on address.

<Frame>
  <img
    src="https://usercontent.in.prod.clueso.io/ebbf25da-f06e-4e91-8b05-b5f8e12127c5/e14daef3-e6cd-4de9-9854-6a86bc00408f/8424db4f-2c21-44ce-9816-315dcf1c3934/images/ba9d25c3-e375-4529-adb7-6edc3d43fb8b.png"
    style={{ borderRadius: "0.5rem" }}
  />
</Frame>

**Step 5:** Input the question you want to ask. For example, "Enter your address".

<Frame>
  <img
    src="https://usercontent.in.prod.clueso.io/ebbf25da-f06e-4e91-8b05-b5f8e12127c5/e14daef3-e6cd-4de9-9854-6a86bc00408f/8424db4f-2c21-44ce-9816-315dcf1c3934/images/de683c75-c30f-4ab7-ab72-3f7ff9327fc4.png"
    style={{ borderRadius: "0.5rem" }}
  />
</Frame>

**Step 6:** If you require the booker to provide an address, turn this option on.

<Frame>
  <img
    src="https://usercontent.in.prod.clueso.io/ebbf25da-f06e-4e91-8b05-b5f8e12127c5/e14daef3-e6cd-4de9-9854-6a86bc00408f/8424db4f-2c21-44ce-9816-315dcf1c3934/images/50760c59-6ba8-430e-a8e2-5acc01b33938.png"
    style={{ borderRadius: "0.5rem" }}
  />
</Frame>

**Step 7:** If the question is optional, keep this option off.

<Frame>
  <img
    src="https://usercontent.in.prod.clueso.io/ebbf25da-f06e-4e91-8b05-b5f8e12127c5/e14daef3-e6cd-4de9-9854-6a86bc00408f/8424db4f-2c21-44ce-9816-315dcf1c3934/images/66524905-75e3-4a86-b650-34b13486d28f.png"
    style={{ borderRadius: "0.5rem" }}
  />
</Frame>

**Step 8:** Click on **Add**.

<Frame>
  <img
    src="https://usercontent.in.prod.clueso.io/ebbf25da-f06e-4e91-8b05-b5f8e12127c5/e14daef3-e6cd-4de9-9854-6a86bc00408f/8424db4f-2c21-44ce-9816-315dcf1c3934/images/77d83d29-54d1-491d-b284-9b9cb0838482.png"
    style={{ borderRadius: "0.5rem" }}
  />
</Frame>

**Step 9:** Your question is now added. If you wish to hide these questions, simply toggle them off.

<Frame>
  <img
    src="https://usercontent.in.prod.clueso.io/ebbf25da-f06e-4e91-8b05-b5f8e12127c5/e14daef3-e6cd-4de9-9854-6a86bc00408f/8424db4f-2c21-44ce-9816-315dcf1c3934/images/ad45e153-13fd-49a5-91e6-c8cd3af3f9bd.png"
    style={{ borderRadius: "0.5rem" }}
  />
</Frame>

<Frame>
  <img
    src="https://usercontent.in.prod.clueso.io/ebbf25da-f06e-4e91-8b05-b5f8e12127c5/e14daef3-e6cd-4de9-9854-6a86bc00408f/8424db4f-2c21-44ce-9816-315dcf1c3934/images/d36acd39-063b-4cb0-8468-c49d549c379b.png"
    style={{ borderRadius: "0.5rem" }}
  />
</Frame>

**Step 10:** You're also able to edit or delete the questions, and make them mandatory if needed.

  <img

  />
</Frame>

</Frame>

<Frame>
  <img

    style={{ borderRadius: "0.5rem" }}

**Step 11:** Click on **Save** to finalize your changes.

<Frame>
  <img
    src="https://usercontent.in.prod.clueso.io/ebbf25da-f06e-4e91-8b05-b5f8e12127c5/e14daef3-e6cd-4de9-9854-6a86bc00408f/8424db4f-2c21-44ce-9816-315dcf1c3934/images/13f3f7c3-8411-42ba-8842-ea395a236d17.png"
