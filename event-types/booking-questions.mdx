---
title: "Booking Questions"
description: "This section will teach you how to add custom questions that bookers must answer when scheduling a time with you."
mode: "wide"
---

**Step 1** 

Start by opening the event type you want to configure.

<Frame><img src="/images/booking_questions/1.webp" /></Frame>

**Step 2** 

Navigate to the **Booking Questions** tab.

<Frame><img src="/images/booking_questions/2.webp" /></Frame>

**Step 3** 

Here, you're able to add or edit your booking questions.

<Frame><img src="/images/booking_questions/3.webp" /></Frame>

**Step 4** 

You can Hide/Unhide questions by enabling/disabling the toggle button.

<Frame><img src="/images/booking_questions/4.webp" /></Frame>

**Step 5**

You can make any question "Mandatory", by enabling "Mandatory" button, After making changes click on "Save" button.

<Frame><img src="/images/booking_questions/5.webp" /></Frame>
